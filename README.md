# 🤖 AI Knowledge Assistant using RAG (Endee)

## 🚀 Overview
This project is an AI-powered Knowledge Assistant built using a Retrieval-Augmented Generation (RAG) approach.

It uses the dataset provided in the Endee repository (README + docs) to perform semantic search and answer user queries.

---

## 💡 Features
- Semantic Search
- RAG-based Retrieval
- Uses Endee dataset
- CLI-based AI assistant
- Clean modular code

---

## 🧠 How it Works
1. Loads dataset from README.md and docs/
2. Cleans and processes text
3. Converts text into tokens (vector-like)
4. Stores in memory (vector store)
5. User query → matched with dataset
6. Returns most relevant answers

- RAG Architecture
