\# 🤖 AI Knowledge Assistant (RAG)



\## 🚀 Overview

This project is a Retrieval-Augmented Generation (RAG) based AI assistant.



It uses dataset from the Endee repository (README + docs) to perform semantic search and answer user queries.



\---



\## 💡 Features

\- Semantic Search

\- RAG-based Retrieval

\- CLI-based interaction

\- Uses real project dataset



\---

git clone https://github.com/Jayashree102/endee.git
